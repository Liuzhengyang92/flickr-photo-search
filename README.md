## How to run the code:

### 1. Clone the code to your local machine useing SSH or HTTPS, or just simply download it.

### 2. Environment Preparation
#### Install node_modules: 
```bash
npm install
```

or

```bash
yarn
```
### 3. Run the code:
```bash
npm start
```

or

```bash
yarn start
```

### 4. Build project

```bash
npm run build
```


### 5. Code test
```bash
npm test
```
